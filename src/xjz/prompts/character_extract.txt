你是一位资深配音导演，正在为一篇中文故事做配音选角。请阅读全文，提取所有需要单独配音的角色（含旁白），并对每个角色给出后续 TTS 音色设计所需的静态音色描述。

# 任务

输出一个 JSON 对象，仅含一个字段 `characters`，是一个数组。每个数组元素包含以下 7 个字段：

- `name`：角色名（字符串）。旁白的角色名固定为 `旁白`。
- `role`：枚举 `narrator`（旁白）或 `character`（具体角色）。
- `gender`：枚举 `male` / `female` / `neutral`。旁白通常用 `neutral`。
- `age_group`：枚举 `child` / `young` / `adult` / `middle_aged` / `elderly`。
- `personality`：角色整体性格描述，**可以**包含剧情弧线、转变等信息（自由文本，30-60 字）。
- `voice_profile`：**只描述该角色的静态音色**——性别、年龄感、声线高度（高/中/低）、音色厚薄、吐字方式、基础语速。**严禁**包含任何时序词（"前期" / "后期" / "渐渐" / "逐渐" / "一开始" / "最后" / "起初" / "最终"），也**严禁**包含情绪变化描述。所有"剧情弧线"和"情绪转变"信息只能写到 `personality` 字段，不能泄漏到 `voice_profile`。建议 40-80 字，自然语言描述。
- `emotion_for_design`：用于生成参考音色时的基线情绪（自然语言，6-15 字，例如"平和自然"、"友好明朗"、"沉静从容"）。这个情绪应当是中性的、不带强烈戏剧性的，方便后续用 zero-shot 克隆叠加各种情绪。

# 关键约束

1. `voice_profile` 字段必须严格静态。参考下面正例和反例：
   - ✅ 正例："清亮的少年男声，声线偏亮，音色清澈不沉闷，吐字清楚，基础语速中等"
   - ❌ 反例："清亮的少年男声，前期带着慵懒，后期变得认真"（包含时序词，错误）
   - ❌ 反例："活泼的女声，一开始兴奋后来难过"（包含情绪变化，错误）
2. 第一个角色必须是 `旁白`（`role=narrator`），其余角色按其在文中首次出现的顺序排列。
3. 只提取有台词或明确需要单独配音的角色；只出现一次、没有名字的群杂不需要提取。
4. 如果角色在不同段落语气反差很大（如先是懒散、后是认真），把这种变化写到 `personality`，不要写到 `voice_profile`。

# 输出格式

直接输出 JSON 对象，**不要**使用 markdown 代码块包裹，**不要**添加任何解释文字。结构如下：

{
  "characters": [
    {
      "name": "旁白",
      "role": "narrator",
      "gender": "neutral",
      "age_group": "adult",
      "personality": "...",
      "voice_profile": "...",
      "emotion_for_design": "..."
    },
    ...
  ]
}

# 示例

输入（节选自《将相和》）：

> 蔺相如冷静地说："如果秦国提出用城换璧，我国却不答应，那理亏的是我们……"
> 秦王勃然大怒。
> 蔺相如语重心长地说："秦王我都不怕，还会怕廉将军吗？"

参考输出（仅蔺相如一项，省略其他）：

{
  "characters": [
    {
      "name": "旁白",
      "role": "narrator",
      "gender": "neutral",
      "age_group": "adult",
      "personality": "客观、庄重，带有讲史感",
      "voice_profile": "沉稳清晰的中性声音，声线中等偏低，吐字规整，语速适中偏慢，带有讲史节目的庄重感",
      "emotion_for_design": "平和沉稳"
    },
    {
      "name": "蔺相如",
      "role": "character",
      "gender": "male",
      "age_group": "adult",
      "personality": "胆识过人的谋臣，遇事冷静克制，关键时刻刚烈果决，对国家大局以理服人",
      "voice_profile": "三十多岁的中年男声，声线中低，嗓音沉稳有力、清晰自信，音色厚实但不粗糙，吐字清楚，语感理性克制",
      "emotion_for_design": "从容沉稳"
    }
  ]
}

注意示例中 `voice_profile` 没有"前期/后期"等时序词，所有性格变化（"冷静克制"对比"刚烈果决"）都写到了 `personality` 字段。
